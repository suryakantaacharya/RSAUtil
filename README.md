# RSAUtil

## Maven Dependency

To use RSAUtil in your Maven project, include the below mentioned dependency in your `pom.xml`:

```xml
<dependency>
    <groupId>io.github.suryakantaacharya</groupId>
    <artifactId>RSAUtil</artifactId>
    <version>LATEST_VERSION</version>
</dependency> ```
